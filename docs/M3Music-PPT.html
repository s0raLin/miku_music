运行 & 展望

快速启动 · 持续进化

🚀 快速启动

  • 环境：Flutter SDK ^3.11.4 · Dart ^3.11.4
  • 安装：flutter pub get
  • 配置：根目录创建 .env（GO_BACKEND_URL=...）
  • 运行：flutter run
  • Rust 编译：自动由 flutter_rust_bridge 处理
  • 支持平台：Android / iOS / Windows / macOS / Linux
